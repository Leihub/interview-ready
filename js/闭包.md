### var let
var 声明的作用域是函数执行的作用域，var 可以穿透if for 语句。
var 有变量提升，会影响多个域。
```
var b = 1;
fn()
console.log(b)
function fn(){
    console.log(b)
    var b = 2
    console.log(b)
}
// undefined //变量提升的undefined
// 2 
// 1

```
let,没有变量提升，必须要在声明后才能使用
```
console.log(b)  // undefined
var b = 1

console.log(a) // 报语法错误
let a = 2
```
### let 暂时死区
在let 和 const 的代码块中，会形成一个封闭区域，在此区域中 如果没有声明变量就是使用，会报语法错误，直到遇到let 声明后死区解除。
```
var a = 1
if(true){
    console.log(a)
    let a = 2
    console.log(a)
}
```
